# Calculator App

A sleek, responsive, and functional web-based calculator built with HTML5, CSS3, and JavaScript. This project features a modern dark-mode UI and handles basic arithmetic operations with precision.

##  Features

* **Basic Operations:** Supports addition, subtraction, multiplication, and division.
* **Advanced Functions:** Includes percentage calculations, sign toggling (+/-), and decimal support.
* **Modern UI:** A clean, dark-themed design using Flexbox for a responsive experience.
* **Smart Display:** Shows both the current expression and the result simultaneously.
* **Backspace & Clear:** Easy correction with a backspace button and a full clear (C) function.
